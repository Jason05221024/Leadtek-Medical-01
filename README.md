# Simple Display Demo

## 📌 Project Overview

This project is a simple HTML-based display screen simulation.

It visually represents a basic LCD-style screen layout using pure HTML and CSS.  
This demo is suitable for:

- GitHub upload practice
- Front-end display layout testing
- Simple UI mockup demonstration
- Basic screen visualization example

---

## 🖥 Preview

The page displays:

- Dark background
- Centered "display module"
- White monospace text
- Framed screen layout

## 🚀 How to Run

1. Download or clone this repository.

4. It will open directly in your web browser.

No installation required.
